# ANN-Mini-Project
# Handwritten Digit Recognition using Artificial Neural Network (ANN)

This project implements an Artificial Neural Network to classify handwritten digits (0–9) using the MNIST dataset.

## Run in Google Colab
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/drive/1euKBb6WwixiWRvfjYC4KjB0eB_J1JFTj?usp=sharing)

## Technologies Used
- Python  
- TensorFlow  
- Keras  
- Google Colab  

## Dataset
MNIST dataset containing:
- 70,000 handwritten digit images
- Image size: 28 × 28 pixels
- Digits from 0 to 9

## Model
ANN architecture:
- Input Layer  
- Hidden Layer (128 neurons)  
- Hidden Layer (64 neurons)  
- Output Layer (10 neurons)

## Accuracy
The model achieves approximately **97–99% accuracy** on the test dataset.
